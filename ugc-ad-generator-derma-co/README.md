# AI UGC Ad Content Generator  
### Case Study: The Derma Co

---

## Overview
This project is an AI-powered prompt system that generates high-converting UGC-style ad content for D2C brands.

It produces:
- Hooks for ad testing  
- UGC-style video scripts  
- CTAs and captions  
- Platform-specific variations (Reels, TikTok, Shorts)  

---

## Problem
Most ads fail because they feel too polished and not relatable, leading to low engagement and conversions.

---

## Solution
A structured AI prompt system that creates authentic, user-like ad content using proven marketing frameworks.

---

## Product Context
**Brand:** The Derma Co  
**Product:** Sali-Cinamide Anti-Acne Serum  

**Target Audience:**  
16–30, oily/acne-prone skin, struggling with acne and low confidence  

---

## Prompting Approach

- **Role-Based Prompting:** AI acts as a performance marketing strategist  
- **Structured Prompting:** Organized input ensures consistent output  
- **Framework Prompting:**  
  Hook → Problem → Agitation → Solution → Result → CTA  

---

## Sample Output

**Hook:**  
I tried everything for my acne… nothing worked.

**Script:**  
I kept switching products but nothing worked.  
Then I tried this serum, and my skin started improving.  
Not instant, but it actually made a difference.  

**CTA:**  
If you’re dealing with acne, try this once.
